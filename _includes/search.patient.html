patient

The patient parameter can be used two ways:

patient=[id]

This SHALL be supported for all endpoint provider systems.

id is the logical id of the patient on the server which is obtained by a Patient resource query.

id SHALL be the national identifier (i.e. NHS Number, CHI Number & HSCNI) for national systems. 

GET /{{include.content}}?patient=42

Return all {{include.content}} resources for Patients with a logical id of 42
patient.identifier={systemUri}|[identifier]

This SHOULD be supported for all non national provider systems for queries by national identifiers (i.e. NHS Number, CHI Number & HSCNI).

This SHALL be supported for all intermediary (e.g. Trust Integration Engines or other middleware) systems.

systemUri is a uniform resource identifier which defines which system the identifer belongs to. For NHS Number this would be https://fhir.nhs.uk/Id/nhs-number and :identifier would be the NHS Number.

GET /{{include.content}}?patient.identifier=https://fhir.nhs.uk/Id/nhs-number|9876543210

Return all {{include.content}} resources for Patients with a NHS Number of 9876543210
