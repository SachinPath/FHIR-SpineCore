patient

The patient parameter can be used two ways:

patient= [id]

id is the logical id of the patient on the server which is obtained by a Patient resource query.

GET /{{include.content}}?patient=42

patient.identifier={systemUri}|[identifier]

:systemUri is a uniform resource identifier which defines which system the identifer belongs to. For NHS Number this would be https://fhir.nhs.uk/Id/nhs-number and :identifier would be the NHS Number. GET /{{include.content}}?patient.identifier=https://fhir.nhs.uk/Id/nhs-number|9876543210

GET /{{include.content}}?patient.identifier=https://fhir.jorvik.nhs.uk/Patient|12345678
